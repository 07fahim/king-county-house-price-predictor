About the Model

Data Source

Based on King County housing data from May 2014 to May 2015, covering 21,613 house sales including Seattle area.

Machine Learning

Powered by a XGBoost regression model that analyzes multiple property features to predict price.

Key Features

Uses factors like square footage, grade, location, bathrooms, and other features to make accurate predictions.
